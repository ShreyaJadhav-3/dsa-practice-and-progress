# Contest Solutions
This folder contains solutions to problems solved during coding contests from platforms like LeetCode and CodeChef.

## Platforms
* LeetCode Weekly & Biweekly Contests
* CodeChef Starters and other contests

## Structure
* Organized by platform and contest name
* Each contest folder contains solutions for that specific contest

## What Each Solution Includes
* Problem details
* Approach used
* Time and space complexity
* Clean and optimized code

## Goal
To improve real-time problem-solving skills, speed, and accuracy by practicing under contest conditions.

## Note
Contest solutions are added regularly to track performance and learning progress.
